TechGear Shop Backend
A robust RESTful API built to power a modern e-commerce platform for high-end technology and hardware. This service handles product catalogs, user authentication, and order processing.

🚀 Getting Started
Prerequisites
Node.js (v18.0.0 or higher)

PostgreSQL or MongoDB (depending on your chosen DB)

Package Manager: npm or yarn

Installation
Clone the repository:

Bash
git clone https://github.com/yourusername/techgear-backend.git
cd techgear-backend
Install dependencies:

Bash
npm install
Set up environment variables:
Create a .env file in the root directory and add your credentials:

Code snippet
PORT=5000
DATABASE_URL=your_database_connection_string
JWT_SECRET=your_super_secret_key
Run the server:

Bash
npm start
